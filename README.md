## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎨 **Dark Emerald Theme** | Premium dark mode with green/emerald accents and glassmorphism effects |
| 🟢 **Interactive Particles** | Canvas-based particle background that reacts to mouse movement |
| ⌨️ **Typing Animation** | Dynamic typing effect cycling through professional titles |
| 📜 **Certificate Gallery** | Certificate cards with images, zoom modal, and verification links |
| 🎯 **Scroll Animations** | Intersection Observer-based reveal animations on scroll |
| 📱 **Fully Responsive** | Works flawlessly on desktop, tablet, and mobile |
| 🔍 **SEO Optimized** | Meta tags, Open Graph, Twitter Cards, sitemap, and Google Search Console verified |
| 📬 **Contact Form** | Functional contact form using `mailto:` protocol |
| ⬆️ **Back to Top** | Smooth scroll-to-top button |
| 🍔 **Mobile Navigation** | Hamburger menu with smooth slide-in animation |

1. **Clone the repository**
   ```bash
   git clone https://github.com/ArnavSharma-2007/ArnavSharma-2007.github.io.git
   ```

2. **Open in browser**
   ```bash
   cd ArnavSharma-2007.github.io
   open index.html
   ```

No build tools, no dependencies — just open and go! ⚡
